# Budget Speeches (Raw Markdown)

This folder contains 66 Singapore budget speeches (1960-2025) in markdown format, extracted from Parliament Hansard.

---

## Overview

Each file represents one budget speech, scraped from the official Parliamentary Reports and converted to clean markdown for processing and analysis.

---

## File Structure

### Naming Convention
Files are named by year: `YYYY.md`

Examples:
- `1960.md` - First budget speech in dataset
- `2025.md` - Most recent speech

### Coverage
- **Total files**: 66
- **Year range**: 1960-2025 (continuous)
- **Total sentences**: 40,123
- **Total words**: ~750,000
